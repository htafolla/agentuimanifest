AgentUIManifest

A standardized JSON schema for defining rich, interactive UI presentations for AI agents and tools.

View Schema GitHub

Core Schema

// AgentUIManifest v1.0.0
{
  "$schema": "https://json-schema.org/draft/2020-12",
  "type": "object",
  "title": "AgentUIManifest",
  "description": "Schema for agent UI presentations",
  
  "properties": {
    "id": {
      "type": "string",
      "description": "Unique identifier"
    },
    "name": { "type": "string" },
    "version": { "type": "string" },
    "description": { "type": "string" },
    "icon": { "type": "string" },
    "color": { "type": "string" },
    
    "presentation": {
      "type": "object",
      "properties": {
        "type": { "type": "string", "enum": ["chat", "dashboard", "form", "canvas"] },
        "layout": { "type": "string" },
        "theme": { "type": "string" },
        "components": { "type": "array" }
      }
    },
    
    "capabilities": {
      "type": "object",
      "properties": {
        "streaming": { "type": "boolean" },
        "tool_calls": { "type": "boolean" },
        "file_upload": { "type": "boolean" },
        "voice": { "type": "boolean" }
      }
    }
  }
}

Core Properties

id string Unique identifier for the manifest
name string Human-readable name
version string Semantic version (e.g., "1.0.0")
description string Brief description of the agent
icon string URL or emoji for the agent icon
color string Primary color (hex or CSS color)
presentation object UI presentation configuration
capabilities object Feature flags and capabilities

Example Manifest

{
  "id": "scout-agent",
  "name": "Scout",
  "version": "1.0.0",
  "description": "Research and analysis agent",
  "icon": "🔍",
  "color": "#6366f1",
  
  "presentation": {
    "type": "dashboard",
    "layout": "research-console",
    "components": [
      { "type": "chat", "position": "left" },
      { "type": "results", "position": "right" }
    ]
  },
  
  "capabilities": {
    "streaming": true,
    "tool_calls": true,
    "file_upload": true
  }
}
